# NexAuth - منصة تسجيل دخول احترافية

  بوابة تسجيل دخول احترافية باللغة العربية مع دعم RTL كامل.

  ## المميزات
  - تسجيل الدخول بالبريد الإلكتروني أو رقم الهاتف
  - تسجيل الدخول عبر Google و Apple
  - واجهة عربية RTL بتصميم داكن عصري
  - استجابة كاملة لجميع الأجهزة
  - مصادقة آمنة عبر Clerk

  ## التقنيات المستخدمة
  - React + Vite + TypeScript
  - Tailwind CSS v4
  - Clerk Authentication
  - Wouter Router
  - pnpm Monorepo

  ## الإعداد
  ```bash
  pnpm install
  pnpm --filter @workspace/auth-portal run dev
  ```
